# NEXORA FIELD AI - MVP V1

Atue como um arquiteto de software sênior especializado em SaaS Marketplace, Field Service Management, Inteligência Artificial e sistemas escaláveis.

Desenvolva a primeira versão funcional da plataforma Nexora Field AI.

Objetivo:

Conectar empresas que necessitam de suporte técnico em campo a técnicos autônomos especializados.

O MVP deve ser focado em validação de mercado e geração de receita.

Tecnologias obrigatórias:

* React
* TypeScript
* Tailwind
* Shadcn UI
* Node.js
* PostgreSQL
* Prisma ORM
* Gemini API

Criar sistema completo com:

# AUTENTICAÇÃO

* Login
* Cadastro
* Recuperação de senha
* JWT
* Controle de acesso

Perfis:

* Administrador
* Empresa
* Técnico
