# CORREÇÃO CRÍTICA — AUTENTICAÇÃO E PERFIL ADMINISTRADOR

Realize uma auditoria completa no sistema de autenticação da plataforma Nexora Field AI.

## Problemas Identificados

### Problema 1

Não é possível criar contas corretamente.

Verificar:

* Cadastro de usuários
* Persistência no banco PostgreSQL
* Prisma ORM
* Validação de formulários
* Criação de sessão
* Hash de senha
* JWT

Corrigir todo o fluxo de registro.

---

### Problema 2

Não é possível acessar o sistema utilizando o usuário administrador.

Verificar:

* Seed inicial
* Tabela Users
* Roles
* Permissões
* Middleware de autenticação
* Controle RBAC

Corrigir login administrativo.

---

ADMIN

ATIVO

Permissões:

* Gestão de usuários
* Gestão de empresas
* Gestão de técnicos
* Gestão financeira
* Dashboard administrativo

---

# Cadastro Público

O perfil ADMIN não deve ser exibido para usuários.

Na tela de cadastro permitir apenas:

* Técnico
* Empresa

Remover completamente:

❌ Administrador

---

# Segurança

Impedir:

* Criação pública de administradores
* Alteração manual de role para ADMIN
* Escalação de privilégios

Somente administradores existentes podem criar novos administradores.

---

# Menu de Cadastro

Antes:

* Técnico
* Empresa
* Administrador

Depois:

* Técnico
* Empresa

Administrador não deve aparecer em nenhuma tela pública.

* E-mail obrigatório

* Recuperação de senha
